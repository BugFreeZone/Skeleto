Документация Skeleto

Добро пожаловать в Skeleto - быстрый и простой Python веб фреймворк

Как начать?

Клонируйте этот репозиторий и в корне выполните: 

pip install .

Пример

from Skeleto import MiniFramework, Response

app = MiniFramework()
def hello():
    return Response("Hello, Skeleto! ")
app.urls['/'] = hello
app.run()
